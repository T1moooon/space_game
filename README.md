# Space Game

Консольная космическая игра в терминале. Управляйте ракетой среди мерцающих звёзд.
Анимированный звёздный фон, звёзды мерцают независимо друг от друга в своей фазе анимации,
движение ракеты ограничено игровым полем.

## Установка и запуск

Требования: Python 3.14+, терминал с поддержкой curses (Linux, macOS или WSL).

Проект использует [uv](https://github.com/astral-sh/uv) как пакетный менеджер:

```bash
uv sync
uv run main.py
```

## Настройки

Параметры игры задаются константами в `main.py`:

- `TIC_TIMEOUT` — задержка между кадрами (по умолчанию `0.1` сек)
- `ROCKET_SPEED` — скорость ракеты в клетках за кадр
- `STARS_COUNT` — количество звёзд на экране
- `BORDER_PADDING` — отступ от рамки
