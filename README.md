# Space Game

Консольная космическая игра в терминале. Управляйте ракетой среди мерцающих
звёзд, уворачивайтесь от космического мусора, а с 2020 года — расстреливайте его
из плазменной пушки.

## Установка и запуск

Требования: Python 3.14+, терминал с поддержкой curses (Linux, macOS или WSL).

Проект использует [uv](https://github.com/astral-sh/uv) как пакетный менеджер:

```bash
uv sync
uv run main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| ↑ ↓ ← → | Движение ракеты |
| Пробел  | Выстрел из плазменной пушки (доступен с 2020 года) |

## Настройки

Параметры игры задаются константами в `main.py`:

- `TIC_TIMEOUT` — задержка между кадрами (по умолчанию `0.1` сек)
- `ROCKET_SPEED` — скорость ракеты в клетках за кадр
- `STARS_COUNT` — количество звёзд на экране
- `BORDER_PADDING` — отступ от рамки
- `TICS_PER_YEAR` — сколько кадров длится один игровой год (по умолчанию `15`, т.е. ≈1.5 сек)
- `PLASMA_GUN_YEAR` — год, с которого доступна стрельба (по умолчанию `2020`)
